== Description ==
Barsi is a Business Consulting WordPress Theme designed for industries, manufacturing, construction, and related sectors. It is based on Underscores and comes with full support for WordPress's block editor and advanced customization features.

== Installation ==
1. Upload the `Barsi` theme folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Themes' menu in WordPress.
3. Navigate to Appearance > Customize to set up your site according to your preferences.

== Frequently Asked Questions ==

= How can I customize the theme? =
You can customize the theme through the WordPress Customizer by going to Appearance > Customize. There, you can change various settings such as colors, menus, logos, and more.

= Does this theme support page builders? =
Yes, Barsi is compatible with popular page builders such as Elementor and WPBakery.
